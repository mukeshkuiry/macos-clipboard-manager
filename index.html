NATIVE MACOS APP

Manage your clipboard history without clutter.

ClipBro is a sleek, ultra-minimalist clipboard manager for macOS. No fancy cloud sync, no tracking, and 100% compliant with your company's security policies.

$ brew install mukeshkuiry/tap/clipboardmanager

Engineered for simplicity and speed.

All of your copied items are indexed instantly, keeping your workflow frictionless.

🔒

Zero-Permission Policy

No macOS Accessibility or Input Monitoring permissions required. 100% compliant with restricted enterprise profiles and IT firewalls.

⚡

Spotlight Overlay UI

Press Option + V to bring up a gorgeous, frosted-glass search overlay instantly at your current mouse cursor position.

🔄

Background Cycling

Press Option + C to cycle backward through clipboard history with an elegant overlay HUD on the bottom of your screen.

Keyboard Cheat Sheet

Action Shortcut
Toggle Clipboard Overlay ⌥ + V
Cycle Clipboard History (HUD) ⌥ + C
Copy Highlighted & Hide Enter / ↩
Instant Copy Row (1 to 9) ⌘ + 1..9
Close Overlay Window Escape / Esc

Quick 1-Minute Install

1

Add the Homebrew Tap

Open your Terminal and tap our official clipboardmanager repository: 

brew tap mukeshkuiry/tap
2

Install the Package

Compile and install the lightweight clipboard manager binary on-the-fly: 

brew install clipboardmanager
3

Start the Service

Register the native launch background daemon to launch automatically at startup: 

brew services start clipboardmanager
